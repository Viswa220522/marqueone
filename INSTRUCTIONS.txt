MARQUE ONE WEBSITE - SETUP AND DEPLOYMENT GUIDE
================================================

This guide provides step-by-step instructions to run, manage, and deploy the Marque One luxury website.

1. RECOMMENDED SETUP (CONDA)
----------------------------
If you use Anaconda or Miniconda, this is the most reliable way to avoid environment issues:
   a. Open your terminal (Anaconda Prompt or PowerShell).
   b. Create a new environment:
      conda create -n marqueone nodejs=20 -y
   c. Activate the environment:
      conda activate marqueone
   d. Install project tools:
      npm install
   e. Run the website:
      npm run dev

2. ALTERNATIVE SETUP (MANUAL NODE.JS)
-------------------------------------
If you prefer not to use Conda:
   a. Install Node.js: Download from https://nodejs.org/ (LTS version).
   b. Restart your terminal (close and reopen it).
   c. Type 'npm install' and then 'npm run dev'.
   d. Access via: http://localhost:3000

3. EXPORTING AND DEPENDENCIES
-----------------------------
(Dependencies are installed automatically via 'npm install'):
   - Next.js, GSAP, Lenis, Framer Motion, Tailwind CSS.

To prepare for production:
   - Build: npm run build
   - Preview: npm run start

4. UPLOADING TO GITHUB
----------------------
   a. Create a repo on GitHub.com.
   b. In your folder:
      - git init
      - git add .
      - git commit -m "Initial cinematic website build"
      - git branch -M main
      - git remote add origin [YOUR_GITHUB_URL]
      - git push -u origin main

5. DEPLOYMENT (MAKING THE SITE LIVE)
------------------------------------
   a. Connect your GitHub account to https://vercel.com/.
   b. Import 'marque-one' and click 'Deploy'.

6. TROUBLESHOOTING & COMMON ISSUES
----------------------------------
ISSUE: "'npm' is not recognized as the name of a cmdlet..."
This means Node.js is installed but your current terminal doesn't know where it is.
   - FIX 1 (Quickest): Close your terminal and open a brand new one. This refreshes the path.
   - FIX 2 (Conda): Ensure you have activated your environment ('conda activate marqueone').
   - FIX 3 (Manual): If using the manual setup, check if "C:\Program Files\nodejs" is in your system Environment Variables.

ISSUE: "Images not loading"
   - Ensure the 'public/hero_images' folder contains the frame sequences.
   - Check if the Marque One logo is in 'public/hero_images/marqueone_logo.png'.

ISSUE: "Animation feels laggy"
   - Ensure you are running in a modern browser (Chrome/Edge recommended).
   - Check if your device has Hardware Acceleration enabled in browser settings.
